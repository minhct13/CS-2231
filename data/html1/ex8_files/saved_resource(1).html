                                                                                                                                    ví dụ:
                                                                                                                                    Pseudo-code for (2:4:2 = 2input:4 neurons:2output)
                                                                                                                                    //'Forward phase
                                                                                                                                    Execute network value (of each neuron)
                                                                                                                                    //'Backward phase
                                                                                                                                    Loop qua mỗi neuron trong network (trừ lớp input)
                                                                                                                                    -đối với output layer, tính delta cho neuron = (yHat - y) * GD
                                                                                                                                    -đối với hidden layer, tính delta neuron = Tổng(delta[] tương ứng ở neuron lớp ouput * weight[] tương ứng ở neuron lớp ouput) * GD
                                                                                                                                    //'GD = (1- y^2) đối với tanH, y * (1 - y) đối với Sigmoid logistic
                                                                                                                                    Loop qua mỗi weight trong neuron
                                                                                                                                    update newWeight[ii] = oldWeight[ii] + LearningRate * Delta * Input[ii]
                                                                                                                                    update newBias[ii] = oldBias[ii] + learningRate * Delta
                                                                                                                                    end loop weight + bias
                                                                                                                                    end loop neuron
                                                                                                                                    ở ví dụ trên mỗi input đầu vào ta update newWeight 1 lần tương ứng cho đến khi loop qua hết input.
                                                                                                                                    nhưng ở mini-batch (ví dụ size=128) ta loop qua 128 inputs mới update newWeight 1 lần, (các ký hiệu anh dùng ở trong bài MLP nó đậm chất toán quá em đọc chữ được chữ mất ) vậy ta cần cộng dồn MSE của mỗi input rồi update ở cuối mini-batch với giá trị = MSE / n ?
